### 1.安装

`npm i -S vue-property-decorator`

### 2.用法

有7个装饰器和1个功能（Mixin）：

- @Prop
- @Prop
- @PropSync
- @Provide
- @Model
- @Watch
- @Inject
- @Provide
- @Emit
- `@Component` (由 vue-class-component 提供)
- `Mixins` (由 vue-class-component 提供)



@Prop：

每个prop的默认值需要定义为与上面显示的示例代码相同。

不支持定义每个默认属性，如@Prop（）prop ='default value'
